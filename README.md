# alp-custom
###Installation
Via bower:<br/>
`bower install alp-custom`


or Download/clone 

include `src/dist/scripts.js` file in your html.
inject 'alpCustom' in your app:

```
angular.module('yourApp', [... , 'alpCustom'])
...
```
